## 🚀 Running the Application

```bash
# Clone and setup
git clone https://github.com/yvh1223/security-analyzer.git
cd security-analyzer
flutter pub get

# Run on web
flutter run -d chrome --web-port 8080

# Application will be available at: http://localhost:8080
```
